# 生命周期说明（占位）

> 对应课程：对象/能力的统一生命周期。循迹小车流程未定，先留骨架。

## 统一生命周期五段（预留）

```
init  →  start  →  process  →  stop  →  deinit
```

| 阶段 | 职责 |
|---|---|
| `init`    | 资源分配、参数加载、进入"已初始化"态（不启动） |
| `start`   | 使能外设/任务，进入"运行"态 |
| `process` | 周期处理 / 事件处理（循迹采样、决策、控制输出） |
| `stop`    | 暂停运行，保留资源 |
| `deinit`  | 释放资源，回到初始态 |
